Class ContextsController
In: controllers/contexts_controller.rb
Parent: ApplicationController

The ContextController imports the general metadata of a dataset. It then is redirected to the ImportController for the data columns (see MeasurementsMethodstep).

Public Instance methods

download()

This action assembles a requested context, or a part of it, at request time. (!! This could be moved to the model with all the private actions, they can then also be used in the ImportController)

edit()

This action provides edit forms for the given context

gather_simple_general_metadata(filename, book)

The general metadata sheet contains information about the data set as a whole. The gather_simple_general_metadata method gathers the contents of the text cells within this sheet. Dates as well as people are collected with other methods (!! read_date_from_workbook, assort_people_to_general_metadata).

index()

There is no index of Context. Instead, the Data index is displayed.

Protected Instance methods

create_categorysheet(book, context, formats, methods = nil)

Creates the fourth sheet of a context file, the one that contains the categoric descriptions. If no Method are given, all appropriate categories of the context will be listed.

create_datasheet(book, context, formats, methods = nil)

Creates the last sheet of a context file, the one that contains the raw data. If no Method are given, all people of the context will be listed.

create_metasheet(book, context, formats)

Creates the first sheet of a context file, the one with the metadata.

create_methodsheet(book, context, formats, methods = nil)

Creates the second sheet of a context file, the one with the method descriptions. If no methods are given, all methods of the context will be used.

create_peoplesheet(book, context, formats, methods = nil)

Creates the third sheet of a context file, the one with the people involved. If no Method are given, all people of the context will be listed.
