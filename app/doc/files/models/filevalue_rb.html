Filevalues manage files. They don‘t contain the file itself, but they point at files located in the /public/files directory. Files are of two types: they can be a raw data spreadsheet, which is uploaded in all it‘s cells and stored in single Measurement instances. They can also be free format files, that are associated with Context instances. !! this link table has still to be done, it may also be added as column to the filevalues table, since every file should be within one context.
